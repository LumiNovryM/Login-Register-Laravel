### === Login-Register-Laravel-Breeze === ### 

# 1. Setup Project 

_1.)Setup File .ENV_

_2.)Install Laravel Breeze, Berikut Perintahnya_ 
`composer require laravel/breeze --dev`
`php artisan breeze:install`
`php artisan migrate`
`npm install`

_3.)Jalankan File Laravel Breeze_
`php artisan serve`
`npm run dev`
